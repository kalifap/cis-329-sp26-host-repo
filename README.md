# Static Website to ExpressJS App

# Express Shopping Cart Assignment

A dynamic web application that converts a static shopping cart into an ExpressJS app using cookies for state management.

## Setup Instructions
1. Clone this repository to your local machine.
2. Open your terminal in the project folder.
3. Run `npm install` to install dependencies (`express`, `cookie-parser`).
4. Start the server by running `node server.js`.
5. Open your browser and visit `http://localhost:3000`.

## Features
- Serves static files (HTML, CSS, JS) via Express.
- Managed shopping cart logic on the backend using Express routes.
- Persists cart data using browser cookies.
